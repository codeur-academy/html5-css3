## D'abord un peu de HTML

````html
<!doctype html>
<html lang="fr">
<head>
    <meta charset="utf-8">
    <title>Démarrer avec CSS</title>
</head>

    <h1>Je suis un titre de niveau un</h1>

    <p>Ceci est un second paragraphe. On y trouve un élément <em>mis en valeur</em>.</p>

    <ul>
        <li>Item un</li>
        <li>Item deux</li>
        <li>Item <em>trois</em></li>
    </ul>

</body>

</html>

````
